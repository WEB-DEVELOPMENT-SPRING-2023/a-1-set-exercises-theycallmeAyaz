HTML & CSS Quick-Reference Guide

HTML Tags CSS Properties Code Examples

HTML Tags:

  • <html>: Defines the root of an HTML document.
  • <head>: Contains metadata about the document (e.g. title, links, scripts).
  • <body>: Contains the visible content of the document (e.g. text, images, videos).
  • <h1> to <h6>: Define six levels of headings, from most important to least important.
  • <p>: Defines a paragraph of text.
  • <a>: Creates a hyperlink to another web page, file, or location within the same page.
  • <img>: Embeds an image into the document.
  • <ul>: Defines an unordered list of items.
  • <ol>: Defines an ordered list of items.
  • <li>: Defines an item in a list.

CSS Properties:

  • background-color: Sets the background color of an element.
  • color: Sets the text color of an element.
  • font-family: Sets the font family of an element.
  • font-size: Sets the font size of an element.
  • text-align: Sets the horizontal alignment of text within an element.
  • text-decoration: Sets the decoration of text within an element (e.g. underline, line-through, none).
  • padding: Sets the padding (i.e. space) inside an element.
  • margin: Sets the margin (i.e. space) outside an element.

Code Examples:

This is an example of using the <p> tag:

<p>This is a paragraph.</p>

This is an example of using the <strong> tag:

<strong>This text is strong.</strong>

This is an example of using the <em> tag:

<em>This text is emphasized.</em>

This is an example of using the <u> tag:

<u>This text is underlined.</u>

This is an example of using the <a> tag:

<a href="https://www.example.com">This is a link to Example.com</a>

This is an example of using the <img> tag:

<img src="example.jpg" alt="Example image">

This is an example of using the <ul> tag:

<ul>
  	<li>Item 1</li>
  	<li>Item 2</li>
  	<li>Item 3</li>
	</ul>

This is an example of using the <ol> tag:

<ol>
  	<li>Item 1</li>
  	<li>Item 2</li>
  	<li>Item 3</li>
	</ol>

This is an example of using the <table> tag:

<table>
  	<tr>
    <th>Header 1</th>
    <th>Header 2</th>
  	</tr>
  	<tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  	</tr>
  	<tr>
    <td>Row 2, Cell 1</td>
    <td>Row 2, Cell 2</td>
  	</tr>
	</table>

This is an example of using the <form> tag:

<form>
  	<label for="name">Name:</label>
  	<input type="text" id="name" name="name"><br>
  	<label for="email">Email:</label>
  	<input type="email" id="email" name="email"><br>
  	<input type="submit" value="Submit">
</form>
